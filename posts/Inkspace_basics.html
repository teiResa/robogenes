Day 2

Following the Inkscape Basics playlist by "Logos by Nick". Today we learned about putting text in a circle. Very useful skill to have.

Wrapping Text Around a Circle

Circle: Use the circle & elipses tool to make a circle. Use ctrl and shift to make it perfectly round. Use the select tool to put it where it needs to go. Click canvas to deselect. Text: Text tool and CAPLOCK to write the text. ALLCAPS is better for readability. Select tool, ctrl, shift to make bigger. Text and Font (the text editor menu), choose font and colour, apply. Wrap it: selection tool, select text, [holding shift] click the circle, Text > Put on Path . Deselect everything. Select the circle twice to get the rotation handles, rotate. Deselect. Select the text, Path > Object to Path. Now the circle can be moved away safely. The text is now naturally in that path shape of a circle. To remove from path: Path > Remove from Path. To Wrap Text within the Circle: Text, shift, circle to select both. Text > Put on Path. Double click for rotation, flip object vertical or horizontally. (triangle flippy buttons) Space the letters if needed using text tool (NOT text menu). Deselect. selection tool to grab text, Path > Object to path. Get rid of circle if you want.

Making Triangles
